# Vairal - AI-Powered Influencer Marketing Platform

## Overview
Landing page for Vairal, an influencer marketing platform. Dark-themed with blue brand accents. Built with React + Express + Vite.

## Recent Changes
- 2026-02-11: Initial landing page build with all sections (hero, features, how it works, stats, platforms, testimonials, pricing, CTA, footer) and login modal with Creator/Brand role selection.

## Project Architecture
- Frontend: React with Tailwind CSS, framer-motion animations, Shadcn UI components
- Backend: Express server (minimal - serves frontend)
- Routing: wouter
- Single landing page at `/`

## Key Files
- `client/src/pages/landing.tsx` - Complete landing page with all sections and login modal
- `client/src/App.tsx` - App router
- `client/src/index.css` - Theme tokens (dark/light mode)
- `client/public/images/` - Generated hero and dashboard images

## User Preferences
- Dark theme with blue brand colors
- Inter font family
- Inspired by captiv8.io, modash.io, influencermarketing.ai, passionfroot.me
